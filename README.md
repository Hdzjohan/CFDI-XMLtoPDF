# CFDI-XMLtoPDF
Bulk transforms CFDI XML to PDF using C#.Net WPF

> **Developed using:**
> - Requires E-Iceblue [Spire.PDF](https://www.e-iceblue.com/Introduce/free-pdf-component.html)
> - QRCoder 1.2.9
